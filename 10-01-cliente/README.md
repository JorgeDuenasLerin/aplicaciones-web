## Importante

¿Dónde se ejecuta php y dónde se ejecuta JS?

```
<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title></title>
    <script type="text/javascript">
      document.writeln("Hola mundo consola!")
      alert('Hola mundo ventana!')
    </script>
  </head>
  <body>
    <h1><?php echo "Hola mundo PHP!!" ?></h1>
  </body>
</html>
```
